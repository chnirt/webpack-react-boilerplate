## Usage

1. Clone repository

```
  git clone https://github.com/chnirt/webpack-react-boilerplate
```

2. Cd into directory

```
  cd webpack-react-boilerplate/
```

3. Install dependencies using npm

```
  npm i
```

4 Start

```
  PORT=<yourport> npm run start
```
